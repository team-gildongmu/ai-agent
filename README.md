# 📌 여행 어시스턴트 (LangGraph + API 라우팅)

사용자의 질문과 현재 위치를 기반으로 한국 관광 API를 호출하고, 대화 이력을 저장하여 개인 맞춤형 여행북을 생성하는 프로젝트입니다.

---

## 📦 프로젝트 실행 방법

### 1️⃣ 프로젝트 클론하기

```
git clone https://github.com/yourusername/travel-assistant.git
cd travel-assistant

```

### 2️⃣ 가상환경(venv) 생성하기

macOS / Linux

```

python3 -m venv venv
source venv/bin/activate

```

Windows

```

python -m venv venv
venv\Scripts\activate

```

### 3️⃣ requirements.txt로 라이브러리 설치하기

```
pip install -r requirements.txt

```

### 4️⃣ 프로젝트 실행하기

```
python src/main.py

```
